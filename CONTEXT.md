# SPEKTOR_V — Project Context

## Brand Overview

**SPEKTOR_V** — бренд дизайнерских украшений из полимерной глины, построенный на личном бренде Виктории Спектор (основатель).

**Позиционирование:** Аутентичное. Тёплые, качественные украшения по доступной цене — "топ за свои деньги". Не премиум. Лесная эстетика: черника, малина, грибы, цветы — природные мотивы.

**История:** 2018 — мастер-классы → эпоксидная смола → полимерная глина → 2022 — Wildberries (~10 SKU) → 2023 — 300+ SKU, выход на Ozon + Яндекс.Маркет → 2024 — 2000+ продаж, уход из образования, полное погружение в бренд.

**Целевая аудитория:** Девушки 16–35 лет. Светлые, добрые, открытые, ценящие натуральное. Любят природу, уют, аутентичность. Чувствительны к истории и атмосфере, не к статусу.

**Маркетплейсы:** Ozon, Яндекс.Маркет, Wildberries  
**Соцсети:** VK (vk.com/spektor_v)

## Site Goal

**Сейчас (MVP):** Брендовый сайт-витрина с редиректом на маркетплейсы.  
**Потом:** Полноценный e-commerce. Архитектура проектируется с прицелом на это.

## Sales Funnel (Главная страница)

```
[Hero — вау-эффект]
    Атмосфера леса + анимированные украшения
    Чёткое позиционирование (кто мы, для кого)
    CTA → Каталог
         ↓
[Топ товаров]
    3–6 лучших украшений с анимацией / параллаксом
    Краткое описание, цена (или диапазон)
    Кнопка: "Купить на Озоне" / "Смотреть на Маркете"
         ↓
[Отзывы]
    Агрегированные отзывы с маркетплейсов
    Рейтинг, фото от покупателей (если есть)
    Социальное доказательство для ЦА 16–35
         ↓
[Краткая история / атмосфера бренда]
    2–3 экрана: кто такая Виктория, откуда лесная тема
    CTA → полная страница "История бренда"
         ↓
[Footer]
    Соцсети, маркетплейсы, копирайт
```

## Visual & UX Direction

**Стиль:** Органичный, тёплый, природный. НЕ минимализм Apple. Вдохновение — лес, ягоды, утренний свет в листве.

**Вау-эффект через:** CSS/JS многослойная анимация поверх качественных фото с маркетплейсов. Параллакс, плавные переходы, интерактивные карточки. НЕ реальные 3D модели (glTF/glb).

**Палитра (направление, уточняется в DESIGN.md):**
- Лесные тона: зелёный (мох, листва), тёплый бежевый, малиновый акцент
- Не стерильный белый — тёплые оттенки молока/льна

**Дизайн-система:** см. [DESIGN.md](./DESIGN.md) (Apple-вдохновлённые принципы, адаптируются под лесную эстетику)

**Анимации:** плавные, органичные (не механические). Эффект "украшение живёт").

## SEO / AEO / GEO

- **SEO:** семантическое ядро вокруг: "украшения из полимерной глины", "украшения лесная тематика", "серьги черника", "подвеска малина" и т.д. Структурированные данные (Schema.org: Product, Review, BreadcrumbList).
- **AEO (Answer Engine Optimization):** FAQ-секции с ответами на вопросы ЦА. Чёткие мета-описания, Open Graph.
- **GEO (Generative Engine Optimization):** Контент написан так, чтобы LLM-поиск (ChatGPT, Perplexity, Яндекс ИИ) цитировал бренд при запросах типа "купить украшения с природными мотивами".

## Technical Architecture

**Текущий этап:** витрина + редирект на маркетплейсы  
**Целевая архитектура:** полноценный e-commerce

**Стек (предварительно, уточняется):**
- Frontend: Next.js (SSR/SSG для SEO) или Astro
- Анимации: GSAP / Framer Motion
- Каталог: CMS (Sanity / Contentful) или хранение в файлах на старте
- Позже: Stripe / ЮKassa для платежей

**Ключевые требования:**
- Быстрая загрузка (Core Web Vitals: LCP < 2.5s) — анимации не должны убивать скорость
- Mobile-first — ЦА 16–35 сидит с телефонов
- Открытая архитектура для добавления e-commerce модуля без переписывания

## Design Strategy

**Этап 1 (сейчас):** Чёрно-белый минимализм Apple. Playfair Display + Montserrat. Фокус на структуру.
**Этап 2:** Добавляем лесную атмосферу — цвета, текстуры, анимации (GSAP).
**Этап 3:** 3D-like эффекты, параллакс, полировка.

**Референс:** Apple.com — отступы, типографика, ритм секций, карточки.
