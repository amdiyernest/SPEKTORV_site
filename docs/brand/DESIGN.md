# SPEKTOR_V — Design System

## 1. Design Philosophy

Минимализм Apple как фундамент. Продукт — герой. Интерфейс отступает. Каждый пиксель служит одной цели: показать украшения так, чтобы захотелось прикоснуться.

**Этап 1 (текущий):** Чистый чёрно-белый. Только структура и типографика.
**Этап 2:** Лесная атмосфера — зелёные акценты, текстуры мха, светлячки.
**Этап 3:** GSAP-анимации, параллакс, 3D-like эффекты на фото.

### Принципы
- Заголовки всегда Playfair Display — это голос бренда
- Всё остальное — Montserrat
- Weight 300 для body создаёт воздушность и лёгкость
- Uppercase только для eyebrow-меток и кнопок
- Центрирование только для заголовков секций. Body-текст — по левому краю

## 4. Layout & Spacing

### Сетка
- Max content width: 1200px
- Padding контейнера: 24px (mobile), 48px (tablet), 64px (desktop)
- Карточки каталога: 4 колонки (desktop), 2 (tablet), 1 (mobile)

### Секции
- Каждая секция: padding-top/bottom 120px (desktop), 80px (mobile)
- Чередование фонов: белый → чёрный → серый → белый (ритм Apple)
- Hero: 100vh, полноэкранная

### Spacing Scale (8px base)
- 4px, 8px, 12px, 16px, 24px, 32px, 48px, 64px, 80px, 120px

## 5. Components

### Navigation (Header)
- Высота: 56px
- Фиксированный (sticky)
- Backdrop: `backdrop-filter: blur(20px); background: rgba(255,255,255,0.72)`
- На тёмных секциях: `background: rgba(0,0,0,0.72)`
- Разделитель снизу: `1px solid rgba(0,0,0,0.06)`
- Лево: текстовые ссылки (Главная, Каталог, Философия бренда, Отзывы)
- Право: кнопка "Связаться" (pill)
- Нет логотипа

### Кнопки

**Primary (pill)**
- Background: `#1d1d1f`
- Color: `#ffffff`
- Padding: 12px 28px
- Border-radius: 50px
- Font: Montserrat, 13px, weight 600, uppercase, letter-spacing 0.08em
- Hover: opacity 0.85
- На тёмном фоне: инвертировать (белый фон, чёрный текст)

**Secondary (outline pill)**
- Background: transparent
- Border: 1px solid `#1d1d1f`
- Color: `#1d1d1f`
- Остальное как primary
- На тёмном фоне: border и color = white

**Text link**
- Color: `#1d1d1f`
- Font: Montserrat, 14px, weight 500
- Hover: underline
- Arrow: ` →` после текста
- На тёмном фоне: color = white

### Карточки товаров

**Размер:** заполняют колонку сетки
**Структура:**
```
┌──────────────────────┐
│                      │
│    Фото (swiper)     │  ← aspect-ratio: 1/1
│    ← • • • →         │  ← индикаторы фото
│                      │
├──────────────────────┤
│ Название товара      │  ← Playfair Display, 18px, 600
│ Краткое описание     │  ← Montserrat, 13px, 400, #86868b
│                      │
│ ★★★★★ 4.9 (42)      │  ← рейтинг + кол-во отзывов
│                      │
│ 490 ₽  ̶6̶5̶0̶ ̶₽̶        │  ← цена + зачёркнутая старая
│                      │
│ [ Купить → ]         │  ← pill button → Telegram
└──────────────────────┘
```
- Background: `#ffffff`
- Border-radius: 16px
- Shadow: `0 2px 12px rgba(0,0,0,0.06)`
- Hover: `translateY(-4px)`, shadow усиливается до `0 12px 40px rgba(0,0,0,0.1)`
- Фото: скруглённые верхние углы, swiper внутри

### Карточки отзывов
```
┌──────────────────────┐
│ ★★★★★                │
│                      │
│ "Текст отзыва..."    │  ← Montserrat, 14px, 300, italic
│                      │
│ ○ Имя                │  ← аватар (инициал) + имя
│   Ozon · 2 дня назад │  ← источник + дата
└──────────────────────┘
```
- На белом фоне: border 1px solid `rgba(0,0,0,0.08)`, radius 16px
- На тёмном фоне: border 1px solid `rgba(255,255,255,0.1)`, bg `rgba(255,255,255,0.04)`

### Footer
- Background: `#000000`
- Padding: 64px
- Текст: `#86868b`
- Ссылки: `#aeaeb2`, hover → `#ffffff`
- Разделитель сверху: `1px solid rgba(255,255,255,0.08)`

## 6. Image Treatment

- Фото украшений: максимальное качество, без фильтров, на чистом фоне или в атмосферной сцене
- На белых секциях: фото с мягкой тенью
- На чёрных секциях: фото "светятся" сами по себе
- Aspect ratio карточек: 1:1 (квадрат)
- Hero: фото на весь экран, объект по центру
- Border-radius на фото: 12px (в карточках — скруглённые верхние углы)

## 7. Responsive Breakpoints

| Breakpoint | Width | Колонки каталога | Nav |
|------------|-------|------------------|-----|
| Mobile | < 640px | 1 | Burger menu |
| Tablet | 640–1024px | 2 | Полная |
| Desktop | > 1024px | 3–4 | Полная |

### Mobile-first приоритеты
- Hero: текст уменьшается, фото остаётся крупным
- Карточки: 1 колонка, полная ширина
- Навигация: бургер-меню
- Footer: вертикальный стек
- Touch targets: минимум 44px

## 8. Do's and Don'ts

### Do
- Щедрые отступы между секциями (120px desktop)
- Фото = герой, текст = поддержка
- Чередовать светлые и тёмные секции для ритма
- Pill-кнопки (50px radius)
- Playfair Display только для заголовков
- Hover-эффекты плавные (0.3s ease)

### Don't
- Нет градиентов (этап 1)
- Нет цветных акцентов (этап 1)
- Нет рамок на карточках (только тень)
- Не больше 2 кнопок в одном блоке
- Не использовать Playfair Display для мелкого текста
- Нет emoji вместо реальных фото (как в первом мокапе)
