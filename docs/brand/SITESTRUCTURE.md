# SPEKTOR_V — Site Structure

## Navigation (Header)

Фиксированный хедер, минималистичный.

| Позиция | Элемент | Действие |
|---------|---------|----------|
| Лево | `Главная` | Ссылка → `/` |
| Лево | `Каталог` | Ссылка → `/catalog` |
| Лево | `Философия бренда` | Ссылка → `/about` |
| Лево | `Отзывы` | Ссылка → `/reviews` |
| Право | `Связаться` (кнопка) | → Telegram (ссылка) |

**Нет логотипа.** Навигация текстовая, чистая.
Стиль: Apple — тонкий, blur-backdrop, чёрный/белый.

## Страница 1: Главная (`/`)

### Секция 1.1 — Hero

**Цель:** Вау-эффект. Первое впечатление = "хочу".

- На весь экран (100vh)
- В центре: украшение на мхе (фото, высокое качество)
- Скролл → запускается параллакс-анимация: украшение "поднимается", фон движется с другой скоростью
- Минимум текста. Возможно: только название `SPEKTOR_V` и одна строка ("Украшения, рождённые в лесу")
- Никаких кнопок в Hero — пусть скроллят
- Scroll indicator внизу (тонкая линия/стрелка)

**Анимация:**
- При загрузке: fade-in украшения (0.8s ease)
- При скролле: многослойный параллакс — украшение, мох, и фон двигаются с разными скоростями
- Эффект глубины без 3D-моделей

### Секция 1.2 — Позиционирование + Короткая история

**Цель:** Кто мы. За 5 секунд.

- Блочная вёрстка: текст слева + фото справа (или наоборот)
- Фото Виктории с 3D-like эффектом (параллакс на hover или scroll)
- Текст: 2–3 предложения, крупный Playfair Display
  - "Каждое украшение лепится вручную"
  - "Вдохновение — в каждой ягоде, в каждом листике"
- Ссылка "Узнать историю →" → `/about`

**Стиль:** чёрный текст на белом фоне, много воздуха, фото с тенью.

### Секция 1.3 — Топ украшений (карточки)

**Цель:** Показать товар. Заставить кликнуть.

- Горизонтальная прокрутка (swiper/carousel) или сетка 3–4 карточки
- Карточки повторяют структуру Ozon:
  - Фото (можно пролистать несколько, swiper внутри карточки)
  - Название
  - Цена
  - Рейтинг (звёзды + число отзывов)
  - Кнопка "Купить" → ведёт в Телеграмм
- Показываем 6–8 карточек, пролистываем
- Внизу: кнопка "Смотреть весь каталог →" → `/catalog`

**Данные:** подтягиваем оценки и отзывы с Ozon, WB, Яндекс Маркет, Золотого яблока, таким образом на нашем ресурсе максимально честная картинка возникает.

### Секция 1.4 — Связаться

**Цель:** CTA перед футером.

- Простой блок: "Чтобы заказать, пишите нам"
- Кнопка "Написать в Telegram" или ВК → ссылка t.me/... и ВК ссылка
- Иконки соцсетей: VK, Telegram (+ другие если есть)
- Минималистично, чёрный фон, белый текст

### Секция 1.5 — Footer

- Навигация (дублирует хедер): Главная, Каталог, Философия бренда, Отзывы
- Ссылки на маркетплейсы: Ozon, Wildberries, Яндекс.Маркет
- Юридические документы:
  - Политика конфиденциальности
  - Публичная оферта
  - Реквизиты ИП / самозанятого
- © 2026 SPEKTOR_V

## Страница 2: Каталог (`/catalog`)

### Структура

**Цель:** Все товары. Удобная навигация. Покупка в один клик (→ маркетплейс).

- **Фильтры** (сверху или сбоку):
  - Тип: серьги, подвески, кольца, броши, наборы
  - Цена: от–до
  - Рейтинг: от 4+
  - Новинки / Хиты / Со скидкой
  - Поиск по названию

- **Сетка карточек** (3–4 колонки desktop, 2 колонки mobile):
  - Каждая карточка = мини-копия карточки Ozon:
    - Фото (swiper, несколько фото)
    - Название товара
    - Цена (старая/новая если скидка)
    - Рейтинг: ★★★★★ (4.9) · 42 отзыва
    - Кнопка "Купить" → выбор маркетплейса (Ozon / WB / ЯМ)
    - Кнопка "Задать вопрос" → Telegram

- **Карточка товара** (при клике на карточку — модальное окно или отдельная страница `/product/:id`):
  - Большие фото (галерея, листаются)
  - Полное описание
  - Характеристики (материал, размер, фурнитура)
  - Отзывы этого товара
  - Кнопки покупки на всех маркетплейсах
  - Похожие товары

- **Пагинация** или бесконечный скролл

**Данные:** Sanity CMS. Можно подтягивать фото с Ozon, хранить в CMS.

### Footer
Стандартный footer (как на главной).

## Страница 3: Философия бренда (`/about`)

### Структура

**Цель:** Показать душу бренда. Вызвать эмоцию. "Хочу быть частью этого".

**Подача:** Сторителлинг. Красивая страница-история. Фотографии чередуются с текстом. Полноэкранные секции.

- **Секция 3.1 — Заглавная**
  - "Как лес стал украшением"
  - Атмосферное фото Виктории за работой (или в лесу)

- **Секция 3.2 — Начало (2018)**
  - Мастер-классы в "Леонардо"
  - "Рисование, бисероплетение, скрапбукинг, декор, флористика, декупаж..."
  - Фото: ранние работы (если есть)

- **Секция 3.3 — Полимерная глина**
  - "Потом начались курсы. Много курсов по лепке"
  - Фото: процесс лепки, руки, инструменты

- **Секция 3.4 — Запуск (2022)**
  - "Были куплены курсы по развитию... за неделю до блокировки"
  - "Я искренне хотела всё бросить, но не сдалась"
  - Первый магазин на Wildberries, ~10 артикулов

- **Секция 3.5 — Рост (2023)**
  - 300+ артикулов, команда, Ozon + Яндекс.Маркет
  - "Работа в образовании для меня закончилась"
  - Фото: команда, упаковка, отправка

- **Секция 3.6 — Сегодня**
  - 2000+ продаж
  - "С нетерпением ждём продолжения этой истории благодаря тебе"
  - CTA → "Выбрать украшение" → `/catalog`

**Стиль:** Каждая секция — полноэкранная. Чередование: фото слева + текст справа, потом наоборот. Scroll-triggered animations (fade-in текста).

## Страница 4: Отзывы (`/reviews`)

### Структура

**Цель:** Социальное доказательство. "Все носят — и я хочу".

- **Статистика сверху:**
  - Общий рейтинг (4.9)
  - Количество отзывов (суммарно с Ozon + WB + ЯМ)
  - Количество продаж (2000+)

- **Лайф-видео:**
  - Карусель / сетка видео с Ozon и VK
  - Видео-отзывы от покупательниц
  - Распаковки
  - Embed с VK / загруженные файлы

- **Текстовые отзывы (карточки):**
  - Сетка карточек 2–3 колонки
  - Каждая карточка:
    - Аватар (инициал)
    - Имя
    - Рейтинг (звёзды)
    - Текст отзыва
    - Фото от покупателя (если есть)
    - Источник: "Ozon" / "Wildberries" / "Яндекс.Маркет"
    - Дата

- **Фильтр:** по маркетплейсу, по рейтингу, по дате

## Дизайн-принципы (текущий этап)

**Стиль:** Чёрно-белый минимализм, вдохновлённый Apple.

| Элемент | Значение |
|---------|----------|
| Основной фон | `#ffffff` (белый) |
| Акцентный фон | `#000000` (чёрный) — Hero, CTA-блок |
| Текст на белом | `#1d1d1f` |
| Текст на чёрном | `#ffffff` |
| Заголовки | Playfair Display, 600–700 |
| Тело текста | Montserrat, 300–400 |
| Кнопки | Чёрные pill (border-radius: 50px) |
| Карточки | Белые, скруглённые углы (16px), мягкая тень |
| Фото | Полноразмерные, без рамок |
| Отступы | Щедрые, Apple-стиль |

**Потом:** добавим лесные цвета, текстуры, анимации. Сейчас — чистая структура.

| SEO | Next.js metadata, Schema.org, sitemap |
